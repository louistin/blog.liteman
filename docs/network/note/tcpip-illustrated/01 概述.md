## 分用

**数据分用**<br>
<img :src="$withBase('/image/network/tcpip-illustrated/01/data_demultiplexing_001.png')" alt="数据分用">

```bash
Demultiplexing

目的主机收到以太网数据帧时, 数据开始从协议栈中由底向上升, 同时去掉各层协议加上报文首部.
  每层协议盒都要检查报文首部中的协议标识, 以确定接受数据的上层协议. 这个过程称作分用.
```
